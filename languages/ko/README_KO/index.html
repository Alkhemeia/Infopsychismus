프로젝트 개요

이 프로젝트는 고전 철학, 현대 과학, 정보 실재론, 범심론을 통합하여 존재와 의식의 본질에 대한 일관된 이론을 형성합니다. 의식적 존재자의 역할과 현실에서 정보의 중요성을 밝히는 것을 목표로 합니다.
