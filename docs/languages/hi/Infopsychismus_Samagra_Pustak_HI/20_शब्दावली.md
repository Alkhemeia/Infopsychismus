# 20. शब्दावली

यह सूची इन्फोप्साइचिज़्म में उपयोग किए गए मुख्य शब्दों को शामिल करती है। उद्देश्य है कि शब्दों की स्पष्ट और संक्षिप्त परिभाषाएँ प्रदान की जाएँ ताकि समझ को आसान बनाया जा सके।
