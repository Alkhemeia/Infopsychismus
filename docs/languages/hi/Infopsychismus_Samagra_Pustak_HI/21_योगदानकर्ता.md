# 21. योगदानकर्ता

यह सूची उन योगदानकर्ताओं के नामों को शामिल करती है जिन्होंने इन्फोप्साइचिज़्म के विकास में योगदान दिया है। इस कार्य में भाग लेने वाले सभी लोगों को विशेष धन्यवाद।
